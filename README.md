# QuickWeb v0.2.2

文档
----------------------------

[http://quick.cnodejs.net]


安装QuickWeb
----------------------------

	npm install quickweb
	
	
开始使用
----------------------------

* 1、创建应用目录结构

	**quickweb -init**
	
* 2、启动应用

	方法一：在应用目录上运行**quickweb**
	
	方法二：**quickweb [应用目录]**
